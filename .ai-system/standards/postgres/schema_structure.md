# SCHEMA STRUCTURE

## Purpose

Definir organização lógica do banco.

---

## Schema Principal

O schema padrão deve ser:

public

---

## Organização por Domínio

Em sistemas grandes, schemas podem ser separados por domínio.

Exemplo:

rh  
financeiro  
estoque

---

## Separação de Responsabilidade

Cada schema deve representar um domínio do sistema.
