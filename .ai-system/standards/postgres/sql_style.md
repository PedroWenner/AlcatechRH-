# SQL STYLE

## Formatação

SQL deve ser escrito de forma legível.

---

## Palavras reservadas

Devem ser escritas em MAIÚSCULO.

Exemplo:

SELECT  
FROM  
WHERE  
JOIN  
ORDER BY

---

## Colunas

Colunas devem ser listadas uma por linha em consultas grandes.

---

## Identação

JOINs devem ficar alinhados para melhorar leitura.
