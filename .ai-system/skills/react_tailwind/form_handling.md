# FORM HANDLING

## Purpose

Definir como formularios devem ser tratados em React.

---

## Principles

- Preferir **React Hook Form** para formulários complexos  
- Validar entradas no frontend antes de enviar para API  
- Mostrar feedback claro de erro para o usuário  

---

## Guidelines

- Inputs devem estar ligados a **state controlado**  
- Mensagens de erro devem ser descritivas  
- Evitar submeter formulários incompletos
