# PERFORMANCE GUIDELINES

## Purpose

Definir práticas de performance para frontend React + Tailwind.

---

## Principles

- Evitar re-renderizações desnecessárias  
- Utilizar memoization quando necessário (React.memo, useMemo, useCallback)  
- Lazy load de componentes e imagens  

---

## Guidelines

- Paginação em tabelas grandes  
- Virtualização de listas  
- Evitar cálculos pesados em render
