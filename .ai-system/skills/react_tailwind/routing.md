# ROUTING

## Purpose

Definir como o roteamento da aplicação deve ser organizado.

---

## Principles

- Utilizar **React Router** para navegação  
- Manter rotas organizadas e centralizadas  
- Separar rotas públicas e privadas  

---

## Guidelines

- Páginas devem ser carregadas dentro de layouts adequados  
- Redirecionamentos devem ser tratados com cuidado  
- Lazy loading de páginas deve ser aplicado sempre que possível
