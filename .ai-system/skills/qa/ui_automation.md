# UI AUTOMATION

## Purpose

Automatizar testes de interface de usuário em aplicações web.

---

## Principles

- Validar elementos visuais, fluxos e interações  
- Cobrir casos críticos e fluxos principais  
- Evitar scripts frágeis dependentes de detalhes de layout  

---

## Guidelines

- Criar scripts robustos usando seletores confiáveis  
- Validar mensagens, alertas, modais e navegação  
- Integrar testes UI no pipeline de CI/CD
