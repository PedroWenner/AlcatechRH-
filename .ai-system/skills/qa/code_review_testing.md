# CODE REVIEW TESTING

## Purpose

Realizar **validação de qualidade em código** durante revisões.

---

## Principles

- Verificar cobertura de testes  
- Validar padrões de código e boas práticas  
- Garantir que novas funcionalidades não quebrem o sistema  

---

## Guidelines

- Revisar scripts de teste automatizados  
- Validar naming conventions e organização de pastas  
- Sugerir melhorias de performance e legibilidade
