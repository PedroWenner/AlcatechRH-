# MANUAL TESTING

## Purpose

Executar **testes manuais** de forma estruturada para validar comportamento da aplicação.

---

## Principles

- Seguir casos de teste documentados  
- Registrar todos os passos executados  
- Identificar e reportar inconsistências  

---

## Guidelines

- Testar funcionalidades críticas primeiro  
- Realizar testes exploratórios para cenários inesperados  
- Validar integração entre sistemas e APIs
