# AUTOMATION SETUP

## Purpose

Configurar **frameworks e ambientes de automação** para testes repetitivos.

---

## Principles

- Automatizar testes críticos e de regressão  
- Separar scripts por tipo de teste: UI, API, performance  
- Integrar com pipelines de CI/CD  

---

## Guidelines

- Usar ferramentas confiáveis (ex: Playwright, Cypress, Selenium)  
- Configurar ambientes isolados para testes  
- Garantir que scripts sejam reusáveis e moduláveis
