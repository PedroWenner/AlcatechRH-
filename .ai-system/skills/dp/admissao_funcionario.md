# ADMISSÃO DE FUNCIONÁRIO

## Purpose

Padronizar todo o processo de **admissão de colaboradores**.

---

## Guidelines

- Validar dados cadastrais (nome, CPF, data de nascimento, CNPJ da empresa)  
- Verificar documentos obrigatórios para admissão  
- Registrar contrato de trabalho com informações de cargo, salário, jornada e regime CLT  
- Gerar alertas para inconsistências antes de enviar para folha  
- Integrar dados com sistema de folha e e-Social
