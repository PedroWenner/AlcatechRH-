# BRANCH CREATION

## Purpose
Criar branches Git padronizadas para cada tipo de solicitação.

## Guidelines
- Seguir convenção:
  - `feature/issue-XX-descricao-curta`  
  - `bugfix/issue-XX-descricao-curta`  
  - `hotfix/issue-XX-descricao-curta`  
- Garantir sincronização com `develop` ou `main` conforme tipo  
- Evitar conflitos e duplicidade de branches
