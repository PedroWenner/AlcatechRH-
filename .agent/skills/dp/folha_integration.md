# FOLHA INTEGRATION

## Purpose

Integrar folha de pagamento com outros sistemas DP e benefícios.

---

## Guidelines

- Integrar com sistema de benefícios (vale-alimentação, transporte, plano de saúde)  
- Sincronizar dados de ponto eletrônico para cálculo de horas extras  
- Garantir que alterações em cadastro impactem automaticamente na folha  
- Registrar logs de integração
