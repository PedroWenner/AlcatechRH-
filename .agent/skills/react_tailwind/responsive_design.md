# RESPONSIVE DESIGN

## Purpose

Definir como a aplicação deve se comportar em diferentes tamanhos de tela.

---

## Principles

- Mobile-first approach  
- Todos os componentes devem se ajustar automaticamente  
- Evitar quebrar layouts em resoluções pequenas ou grandes  

---

## Guidelines

- Utilizar **Tailwind responsive utilities**  
- Verificar comportamento em mobile, tablet e desktop  
- Evitar scroll horizontal desnecessário
