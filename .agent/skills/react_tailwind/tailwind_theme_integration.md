# TAILWIND THEME INTEGRATION

## Purpose

Definir como integrar TailAdmin Free Theme com a aplicação React.

---

## Principles

- Utilizar classes utilitárias do Tailwind  
- Respeitar cores, tipografia e espaçamentos do template  
- Evitar sobrescrever estilos base do TailAdmin sem necessidade  

---

## Guidelines

- Componentes novos devem herdar layouts existentes  
- Evitar duplicação de classes e estilos  
- Manter consistência visual em toda a aplicação
