# CODING STANDARDS

## Purpose

Definir padrões de código para React + Tailwind.

---

## Principles

- Código limpo e legível  
- Nomes descritivos de componentes, props e funções  
- Separação clara de responsabilidades  

---

## Guidelines

- Usar camelCase para props e variáveis locais  
- Usar PascalCase para componentes  
- Evitar lógica complexa diretamente em JSX
