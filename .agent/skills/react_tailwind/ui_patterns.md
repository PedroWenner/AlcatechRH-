# UI PATTERNS

## Purpose

Definir padrões visuais e de interação para a aplicação.

---

## Principles

- Seguir **TailAdmin Free Theme** como base  
- Componentes devem ser consistentes com o design system  
- Evitar estilos inline excessivos  

---

## Guidelines

- Botões, inputs e tabelas devem seguir padrões de tema  
- Componentes reutilizáveis devem respeitar responsividade  
- Evitar duplicação de componentes ou padrões visuais
