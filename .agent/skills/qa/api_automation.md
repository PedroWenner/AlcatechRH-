# API AUTOMATION

## Purpose

Automatizar testes de APIs para validar integração entre sistemas.

---

## Principles

- Cobrir endpoints críticos e fluxos de negócio  
- Validar respostas, códigos de status e dados retornados  
- Testar casos de sucesso e erro  

---

## Guidelines

- Usar ferramentas como Postman, REST Assured ou Axios + Jest  
- Criar scripts que possam ser integrados em CI/CD  
- Evitar duplicação de testes e scripts
