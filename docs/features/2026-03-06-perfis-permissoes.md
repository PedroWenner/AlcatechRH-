# Feature: Gerenciamento de Perfis e Permissões

## Visão Geral
Módulo para cadastrar, editar, ativar e excluir novos perfis e associar permissões a eles.

## Requisitos
- Interface padronizada (usando componentes genéricos `Table`, `Modal`, `FormInput`).
- Listagem de perfis com paginação.
- Ações: Cadastrar novo, Editar existente, Ativar/Inativar, Excluir.
- Seleção de permissões por perfil.
