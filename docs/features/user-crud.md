# Feature: User Management CRUD

## Description
Create a UI for User Registration (CRUD). An endpoint for user registration already exists, but a user interface is missing to view, create, edit, and delete users along with their profile selection.

## Classification
- Type: Feature
- Component: Frontend (DPManagement.Web)
- Areas: Pages (Users), Services, AppLayout 

## Requirements
1. Interface to list existing users.
2. Interface to create a new user (including Profile Selection).
3. Interface to update an existing user's details and profile.
4. Interface to delete a user.
5. Integration with the existing registration endpoint.
