# Feature: Dados Bancários

## Description
Provide a system to register and manage multiple Bank Details (Dados Bancários) for each Employee (Colaborador).

## Requirements
- Relationship: 1 Colaborador -> N Dados Bancários.
- Fields required for a Bank Detail:
  - Código do Banco
  - Agência
  - Dígito da Agência
  - Conta
  - Dígito da Conta
  - Operação
- Establish a user interface allowing the creation, reading, updating, and deletion of bank records linked to employees.
