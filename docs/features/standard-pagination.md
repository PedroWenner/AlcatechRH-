# Feature: Standard Pagination

## Description
Adicionar um sistema de paginação padronizado para todas as tabelas do sistema, garantindo performance e uma melhor experiência de usuário.

## Requirements
- Paginação no backend (Skip/Take).
- Componente de paginação reutilizável no frontend.
- Identidade visual moderna e responsiva.
- Suporte a mudança de página e carregamento assíncrono.

## Technical Details
- **Backend**: Update repositories and services to return paginated data.
- **Frontend**: Create `Pagination.tsx` component.
- **Pages**: `Employees.tsx`, `Cargos.tsx`.
